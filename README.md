d# Bahar Lawter Portfolio

Portfolio website for Bahar Lawter, former educator transitioning to software engineering.

## Project Structure

## Page Organization

### Main Pages (in `/pages/`)

- **`about.md`** - Personal story, career journey, and background
- **`contact.md`** - Contact information and social links
- **`projects.md`** - Showcase of development projects
- **`resume.md`** - Professional resume and experience
- **`index.md`** - Homepage and highlights

### Content (in `/content/`)

- **`posts/`** - Blog posts about the journey, learning experiences, and insights

## Technologies Used

- **Jekyll** - Static site generator
- **HTML/CSS** - Structure and styling
- **Markdown** - Content creation
- **GitHub Pages** - Hosting

## Features

- Responsive design
- Professional portfolio showcase
- Blog for sharing journey and insights
- Contact information and social links
- Clean, organized file structure

## Development Workflow

1. **Content Creation**: Write new pages in `/pages/` or blog posts in `/content/posts/`
2. **Jekyll Processing**: Jekyll generates the site in `/docs/_site/`
3. **GitHub Pages**: Automatically deploys from the About branch
